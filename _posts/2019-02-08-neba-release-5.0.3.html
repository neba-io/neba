NEBA 5.0.3 released

NEBA 5.0.3 features an improved handling of tailing vs. following a logfile and allows for easier selection of logfiles.
Furthermore, the names of the CRX packages have been updated to reflect whether the NEBA core or NEBA core + spring delivery package has been installed in Adobe AEM™.
Finally, the optional NEBA Spring integration has been updated to the current Spring 5 GA version 5.1.4 containing multiple important buf fixes and improvements.
