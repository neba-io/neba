NEBA 5.0.7 released

NEBA 5.0.7 ships with minor bugfixes for the log viewer's scroll back buffer and better handling of missing access to the JCR repository in the model registry console

Finally, the optional NEBA Spring integration has been updated to Spring Version 5.1.8, containing multiple important bugfixes and improvements.
