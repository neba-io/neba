NEBA 5.0.1 released

In 5.0.1, NEBA updates to the current Spring GA release 5.0.8 and ships with minor improvements and bug fixes.

      Upgrading from 5.0.0

      Users of 5.0.0 do not need to make any changes. However, if you are using NEBA's Spring integration, we recommend upgrading from Spring 5.0.7 to Spring 5.0.8.
